# README - Meeting Memo Generation

> **Last Updated**: 2026-02-25 by Keming He

Generate meeting memos that capture decisions, actions, and key discussions following project templates.

## Quick Start

Use this skill when you need to:

- Document meeting outcomes and decisions
- Create action item lists with ownership
- Write daily standup summaries

Tell your AI agent: "Create a meeting memo" or "Document this meeting".
